# Minimal GitHub Pages — Academic Site

This repo is a click-only starter for a one-page academic profile using GitHub Pages with the **Minimal** theme.

## Publish
1. Create a repo named **YOURUSERNAME.github.io** on GitHub.
2. Upload these files to the repo.
3. Go to **Settings → Pages** and ensure Source = `Deploy from a branch`, Branch = `main`.
4. Visit `https://YOURUSERNAME.github.io`.

## Customize
- Edit `index.md` directly on GitHub.
- Replace the logo (optional): `portrait.jpg` in the repo root.
- Update publications and links.
